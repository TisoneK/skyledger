## Why

The SkyLedger foundation is complete and functional, but users are experiencing visual friction points that reduce clarity and usability. These issues violate the core philosophy of "Clarity over Complexity" and impact the user's ability to quickly understand their financial data at a glance.

## What Changes

- **Role Performance Cards**: Fix text overlapping and data collision between labels and currency values
- **Chart Visual Improvements**: Add missing legends and reduce gridline opacity for better readability
- **Action Button Hierarchy**: Establish visual priority between primary and secondary actions
- **Layout Optimization**: Move Quick Stats from sidebar to main dashboard area for better visibility
- **Accessibility Enhancements**: Improve contrast and readability in Chama Target Tracker
- **Cleanup**: Remove temporary Tailwind test placeholder

## Capabilities

### New Capabilities
- `role-card-layout`: Improve spacing and visual hierarchy in role performance cards
- `chart-visual-enhancements`: Add legends, tooltips, and soften gridlines
- `action-hierarchy`: Establish primary/secondary button styling patterns
- `dashboard-layout-optimization`: Reorganize sidebar and main content layout
- `accessibility-improvements`: Enhance contrast and readability across components

### Modified Capabilities
- `weekly-dashboard`: Update layout structure to accommodate Quick Stats repositioning

## Impact

**Affected Components:**
- Role Performance Cards (Personal, Sky Tech, Chama, Side Income)
- Weekly Trends Charts (Income vs Expenses, Net Income Trend)
- Quick Actions section (Add Transaction, View Analytics, Export Report)
- Sidebar navigation and Quick Stats
- Chama Target Tracker progress display

**Design System Changes:**
- Establish button hierarchy patterns (primary vs secondary)
- Define chart legend and tooltip standards
- Update spacing guidelines for data display cards

**User Experience Impact:**
- Improved data readability and reduced cognitive load
- Clearer visual hierarchy for user actions
- Better accessibility compliance
- More efficient use of screen real estate
